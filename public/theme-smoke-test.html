Smoke Test Verification Checklist

Buttons
  • Primary button matches theme color
  • Text is readable on all button types
  • Hover states work correctly
Badges & Alerts
  • Status colors are distinct
  • Text contrast is sufficient
  • Light backgrounds in light theme, dark in dark
Forms
  • Input backgrounds match theme
  • Focus ring uses primary color
  • Placeholder text is visible but muted
Header/Footer
  • Logo uses primary color
  • Nav links have proper hover
  • Footer text is readable
Cards & Dashboard
  • Card backgrounds contrast with page
  • Stat cards use theme light colors
  • Borders are visible but subtle
Overall
  • Both themes are visually cohesive
  • No jarring color transitions
  • All text is readable
